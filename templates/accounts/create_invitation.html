How Provider Invitations Work

For You (Administrator)

  • • Enter the provider's email address and select their role
  • • Add an optional personal message
  • • Set expiration period (1-30 days)
  • • Track invitation status from the management dashboard

For the Provider

  • • Receives a secure email with invitation link
  • • Clicks link to access registration form
  • • Creates account with pre-assigned role
  • • Gains immediate access to appropriate features
